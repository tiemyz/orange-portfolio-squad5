## ⚙️ Rodando Localmente

Instale o Orange Portfólio com npm, seguindo os seguintes passos:

- Baixar a node modules:

```bash
npm install
```

- Instalar o firebase:

```bash
npm install firebase
```

- Rodar o projeto:

```bash
npm run dev
```
